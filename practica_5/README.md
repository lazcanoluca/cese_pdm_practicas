En esta práctica se implementa un debouncing por software utilizando una máquina de estados finitos. Se utiliza una versión levemente modifica del módulo `delay` implementado en prácticas anteriores, y un módulo `debounce` implementado en esta práctica.

En `main`, luego de inicializar el debouncing, se actualiza en cada iteración del `while` y prendiendo o apagando el LED 2 en consecuencia.
